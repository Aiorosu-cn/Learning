# 做动画

相当于给UI做动画

创建一个UI后,下面有timeline

首先需要选择一个动画,那么首先需要创建一个动画,左边创建动画

选择动画后,右边的窗口就激活了

点击轨道`track`,可以添加事件(触发或者重复),下面是材质参数,上面的轨道是和控件相关

红色轨道是指定了播放的事件

点击钥匙,开启录制窗口
